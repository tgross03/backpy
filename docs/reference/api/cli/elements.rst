**********************************
Elements (``backpy.cli.elements``)
**********************************

.. currentmodule:: backpy.cli.elements

The :mod:`backpy.cli.elements` module contains functions and classes
used to interact with the :ref:`CLI <cli-reference>`.

The contained classes are used to prompt the user to enter values (e.g. in :ref:`interactive-mode`) or
to confirm a command.

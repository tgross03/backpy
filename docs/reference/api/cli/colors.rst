******************************
Colors (``backpy.cli.colors``)
******************************

.. currentmodule:: backpy.cli.colors

The :mod:`backpy.cli.colors` submodule contains the color palettes used in the :ref:`CLI <cli-reference>`.
It uses the color palettes from `Catppuccin <https://catppuccin.com/>`_ and translates them into ANSI codes
that can be displayed as colors in the terminal.
